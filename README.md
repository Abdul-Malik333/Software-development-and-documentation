# Smart Hostel Management System

This project is a Smart Hostel Management System developed as part of the Software Engineering course.

## Project Description
The system helps manage hostel operations including:
- User management (students and admins)
- Room management
- Booking and approvals
- Payments
- Maintenance requests

## Technologies
- Jira for project management
- GitHub for version control
- UML for system design
- Database (ER Diagram)

## Project Structure
- Documentation: Contains all design and analysis documents
- SourceCode: Contains implementation code (Part 3)
